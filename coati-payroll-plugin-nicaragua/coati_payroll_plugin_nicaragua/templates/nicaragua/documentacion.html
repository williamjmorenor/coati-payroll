Conceptos Principales
INSS (Instituto Nicaragüense de Seguridad Social)
  • Deducción del trabajador: 7% del salario bruto
  • Se aplica antes del cálculo del IR
  • Base de cálculo: salario bruto mensual
IR (Impuesto sobre la Renta)
  • Sistema progresivo por tramos
  • Método acumulado (considera ingresos del año fiscal)
  • 5 tramos fiscales: 0%, 15%, 20%, 25%, 30%
  • Base de cálculo: salario bruto menos INSS
Cálculo Paso a Paso
  1. Calcular salario bruto (salario base + percepciones)
  2. Aplicar INSS (7% del bruto)
  3. Calcular base para IR (bruto - INSS)
  4. Acumular ingresos del año fiscal
  5. Calcular IR anual acumulado según tabla progresiva
  6. Restar IR pagado en meses anteriores
  7. Resultado: IR del mes actual
Configuración
Inicializar Plugin
payrollctl plugins nicaragua init

Carga los catálogos base (deducciones, tipos de planilla, etc.)

Actualizar Plugin
payrollctl plugins nicaragua update

Aplica actualizaciones al catálogo

Responsabilidad

El implementador debe:

  • Validar resultados manualmente
  • Mantener actualizado el sistema
