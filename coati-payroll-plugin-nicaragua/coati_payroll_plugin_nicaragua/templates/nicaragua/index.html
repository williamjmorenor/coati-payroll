Plugin de Nicaragua

Funcionalidades específicas para cálculo de nómina en Nicaragua

Cálculos de Nómina
INSS (Instituto Nicaragüense de Seguridad Social)

Deducción del 7% sobre el salario bruto.

IR (Impuesto sobre la Renta)

Cálculo progresivo con método acumulado según tabla de tramos fiscales.

Los cálculos se realizan según la legislación nicaragüense vigente y deben ser configurados por el implementador.

Utilidades
Scripts de Validación

El plugin incluye scripts para validar los cálculos:

python scripts/validar_ejemplos_nicaragua.py
python scripts/reporte_validacion_nicaragua.py
Tests Automatizados

Suite completa de tests para validar cálculos de IR e INSS:

pytest tests/test_nicaragua_ir_calculation.py
pytest tests/test_nicaragua_json_validation.py
Importante

Este plugin sigue el Contrato Social de Coati Payroll:

  • No incluye interpretaciones legales vinculantes
  • No reemplaza conocimiento profesional
  • No garantiza cumplimiento regulatorio

El implementador es responsable de configurar correctamente los cálculos según la legislación nicaragüense vigente y validar los resultados.
